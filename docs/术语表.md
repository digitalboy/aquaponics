# 连锁数字化农业工厂：鱼菜共生多学科术语表 (Glossary)

鱼菜共生数字化工厂项目横跨**水产养殖、设施园艺、工业自动化 (IIoT)、具身智能 (AI) 以及供应链财务**五个截然不同的学科领域。为了使集团总部、各基地现场（养殖员、农艺师、电工）与 IT 研发团队之间达成“语言一致，认知对齐”，特制定本术语表。

---

## 1. 水产养殖与生态生物学领域 (Aquaculture & Biology)

| 术语 / 缩写 | 英文全称 / 原理 | 中文定义与商业/技术关联 |
| :--- | :--- | :--- |
| **TAN** | Total Ammonia Nitrogen | **总氨氮**：水体中非离子氨 ($\text{NH}_3$) 与离子铵 ($\text{NH}_4^+$) 的总和。鱼类排泄产生的 $\text{NH}_3$ 具有极强毒性（$>0.1\,\text{mg/L}$ 即可引发窒息死亡）。是系统最核心防范的生化毒素指标。 |
| **DO** | Dissolved Oxygen | **溶解氧**：溶解在水体中的氧气量。鱼池一般需 $>5.0\,\text{mg/L}$，强耗氧的硝化反应器需 $>6.0\,\text{mg/L}$，是生物维持生命的第一安全红线，PLC 必须做无条件硬件硬锁闭保护。 |
| **FCR** | Feed Conversion Ratio | **饲料转化率 (料肉比)**：增加一单位鱼体重量所消耗的饲料重量之比。FCR 越低，说明饲料吸收效率越高。是控制水产养殖中最大 OPEX（饲料成本）的关键绩效指标 (KPI)。 |
| **RAS** | Recirculating Aquaculture System | **循环水产养殖系统**：通过重力微滤机（物理过滤）、生物滤池（硝化反应器）等设备将水体净化循环利用的封闭式陆基养殖系统。是鱼菜共生“鱼端”的物理载体。 |
| **硝化反应** | Nitrification | **硝化作用**：由亚硝化单胞菌与硝化杆菌协同作用，将有毒的总氨氮 (TAN) 氧化为无毒的硝酸盐 ($\text{NO}_3^-$) 的生化过程。该过程为强耗氧并强释放氢离子的过程，会直接导致水质 pH 下降。 |
| **EC** | Electrical Conductivity | **电导率**：反映水溶液中总可溶性盐浓度的物理量。在水培“菜端”作为营养液浓度的代理指标，用以控制加肥机的配比步长。 |

## 2. 现代种植与环控领域 (Hydroponics & Environment)

| 术语 / 缩写 | 英文全称 / 原理 | 中文定义与商业/技术关联 |
| :--- | :--- | :--- |
| **VPD** | Vapor Pressure Deficit | **饱和蒸汽压差**：特定温度下饱和空气水汽压与实际水汽压的差值（单位 $\text{kPa}$）。是评价植物蒸腾作用、养分输送效率的黄金指标。VPD 过高或过低都会引发叶片气孔闭合或烧心病，环控算法需秒级动态修正此指标。 |
| **PPFD** | Photosynthetic Photon Flux Density | **光合有效辐射通量密度**：在光合有效波长范围内，每秒照射到每平方米表面的光子数（单位 $\mu\text{mol}/(\text{m}^2\cdot\text{s})$）。是评估温室光照强度的指标，作为 AI 调控 LED 补光灯的直接输入特征。 |
| **DLI** | Daily Light Integral | **日光照累积量**：植物在 24 小时内每平方米接收到的光合有效光子的累积量。AI 会结合未来天气预报，对 DLI 进行天级预算调配，以实现策略性省电。 |
| **NDVI** | Normalized Difference Vegetation Index | **归一化植被指数**：利用植物叶绿素吸收红光并反射近红外光的原理计算的健康指数。巡检机器人通过搭载高光谱相机捕捉此数据，以在肉眼观察到黄叶的 **48 小时前**检测出蔬菜的养分亏缺或潜伏期病害。 |
| **VPD 反演** | VPD Inversion | **基于环境的反推**：在现场未布置叶片温度传感器时，利用温室内温湿度传感器数据结合局部流体力学空气阻力模型，反推出蔬菜叶片表面的 VPD 指标。 |

## 3. 物联网、边缘与自动化控制领域 (IIoT & Automation)

| 术语 / 缩写 | 英文全称 / 原理 | 中文定义与商业/技术关联 |
| :--- | :--- | :--- |
| **PLC** | Programmable Logic Controller | **可编程逻辑控制器**：工业级底层控制器（如西门子S7）。运行嵌入式梯形图程序，负责执行水泵、变频器、热泵的实时开关逻辑。其闭环保护逻辑必须脱离操作系统和网络独立工作（确保高鲁棒性）。 |
| **Edge Gateway** | Intelligent Edge Gateway | **智能边缘网关**：部署于温室现场的本地工控机，处理高吞吐量视频流推理（如鱼群抢食分析），支持本地 TSDB 缓存、断网续传，实现本地数据在断网状态下的“边缘自治”。 |
| **时空同步时间戳** | Spatiotemporal Timestamping | **时空同步标签**：一种数据对齐机制。考虑到水循环在管道及反应器中存在空间物理流动和时间滞后性，算法将 $T_0$ 时刻菜池的数据与 $T_0 - 35\,\text{min}$ 前对应的鱼池投喂数据进行因果关系解耦对齐。 |
| **Soft Sensor** | Software / Virtual Sensor | **软传感器 (虚拟传感器)**：在现场缺乏高昂物理传感器（如氨氮实时化验仪）或探头极易漂移时，利用其他易测的“硬数据”（如 pH 变化斜率、耗氧率速度）建立多变量数学估算模型，实时计算出当前不可见生化指标的数值。 |
| **QoS 0/1/2** | Quality of Service (MQTT) | **MQTT服务质量等级**：QoS 0（发送即可，用于高频温湿度）；QoS 1（保证至少送达一次，用于设备开关反馈）；QoS 2（保证仅送达一次，用于死淘汰换等资产流转数据）。 |

## 4. 机器人与具身智能领域 (Robotics & Embodied AI)

| 术语 / 缩写 | 英文全称 / 原理 | 中文定义与商业/技术关联 |
| :--- | :--- | :--- |
| **RCS** | Robot Control System | **机器人调度与集中控制系统**：多基地的“交警总指挥”。负责将 ERP 的业务工单（如“收割C区生菜”）拆解并派发为 AMR 地面行驶路线、天车避让时段与机械臂收割工单的任务链。 |
| **AMR** | Autonomous Mobile Robot | **自主移动机器人**：搭载激光雷达、双目避障相机和 SLAM 算法的地面小车，负责在温室中运输重达数百公斤的水培蔬菜浮板。 |
| **Embodied AI** | Embodied Artificial Intelligence | **具身智能**：AI 算法与物理躯体结合的技术。如六轴收割机械臂利用 3D 深度相机抓取生菜点云，并进行实时柔性轨迹生成，实现在不压碎菜叶的前提下抓取并切根。 |
| **3D 点云识别** | 3D Point Cloud Processing | **三维空间重构**：3D 相机发射结构光或利用双目视差获取的物体表面海量空间坐标点。AI 系统通过解析点云数据来重构鱼的体型计算重量，或定位生菜根茎的精细抓取中心。 |

## 5. 财务与供应链管理领域 (Finance & Supply Chain)

| 术语 / 缩写 | 英文全称 / 原理 | 中文定义与商业/技术关联 |
| :--- | :--- | :--- |
| **CAPEX** | Capital Expenditure | **资本性支出 (固定资产投资)**：新建园区、购买连栋温室主体、自动化育苗流转线、工业 PLC 柜和机器人设备的初期一次性建造成本。 |
| **OPEX** | Operating Expense | **运营性支出 (日常开支)**：农场运行所需的变动成本，核心包括电费（水泵/暖通/补光）、鱼饲料费、人工工资以及日常易耗传感器探头的更换费。 |
| **ROIC** | Return on Invested Capital | **投资资本回报率**：评估数字化工厂赚钱效率的终极财务指标。IT 系统的 MPC 能耗优化、机器人降本都是为了提升此比率。 |
| **TOU** | Time-of-Use Pricing | **峰谷分时电价**：根据电力市场尖峰、平段、谷段计价的电费标准。AI 系统（MPC 算法）必须强力接入电网分时电价表，实现在深夜蓄热蓄能对冲白天的高 OPEX 阶梯电价。 |
| **资产周转率错位** | Asset Turnover Mismatch | **长慢周期与快短周期冲突**：生菜 30 天即可收割一茬并回笼现金，但三文鱼/鳕鱼需要 12-18 个月方可上市。财务与供应链模块需动态计算此错位，确保集团现金流永不断流。 |
| **OMS** | Order Management System | **订单管理系统**：用于承接连锁大商超（如盒马）的大宗销售订单，并反向驱动各连锁数字化基地进行精细排产。 |
