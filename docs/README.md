## 🛠️ 数据规范与安全红线（【全局防护墙】）

> [!IMPORTANT]
> 凡是向系统贡献代码或进行二次设计的开发者，必须遵循以下核心约束：
> 1. **时钟必须对齐**：所有现场的 PLC、边缘网关与传感器时钟，必须通过 NTP 每小时强制同步，时差超过 $100\,\text{ms}$ 将触发系统熔断。
> 2. **数据冷热隔离**：高频时序遥测数据保留 30 天，30 天后由降采样引擎压缩归档大容量冷库。
> 3. **人身与生物安全**：现场部署的 AMR 无人车必须配置超声波防撞与物理碰条，PLC 逻辑的“应急曝气”逻辑高于一切上层 AI 指令。
