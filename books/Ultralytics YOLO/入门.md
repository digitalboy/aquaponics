**Ultralytics YOLO系列初级入门知识框架（20条）**，帮助你快速建立认知。YOLO（You Only Look Once）是实时目标检测的代表，Ultralytics（从YOLOv5开始）将其做成易用、强大的PyTorch框架，目前主流是**YOLOv8、YOLO11**（YOLOv11比v8参数更少、精度更高）。

1. **YOLO核心思想**：单阶段检测器，一次前向传播同时预测边界框（bounding box）、类别和置信度，速度远超两阶段方法（如Faster R-CNN），适合实时场景。

2. **Ultralytics YOLO**：Ultralytics公司维护的开源实现（GitHub: ultralytics/ultralytics），从YOLOv5开始用PyTorch重构，极大简化使用。支持YOLOv8、YOLO11、YOLO26等版本。

3. **主要任务（Tasks）**：不止物体检测，还支持实例分割（Segment）、图像分类（Classify）、姿态估计（Pose）、定向边界框（OBB）、语义分割等。多任务统一框架。

4. **模型规模变体**：n（nano，最小最快）、s（small）、m（medium）、l（large）、x（extra large）。n/s适合边缘设备，l/x追求最高精度。

5. **安装简单**：`pip install ultralytics`，一行代码即可使用。自动下载预训练权重（.pt文件）。

6. **CLI命令行使用**：核心用法 `yolo task=detect mode=predict model=yolo11n.pt source=image.jpg`。支持train/val/predict/export/track等模式。

7. **Python API**：更灵活，例如 `from ultralytics import YOLO; model = YOLO('yolo11n.pt'); results = model('image.jpg')`。

8. **五大核心模式（Modes）**：
   - **Train**：训练/微调模型。
   - **Val**：验证模型性能（mAP等指标）。
   - **Predict**：推理/预测。
   - **Export**：导出为ONNX/TensorRT/OpenVINO等部署格式。
   - **Track**：多目标跟踪（视频）。

9. **数据集格式**：检测任务常用YOLO格式（每行：class x_center y_center width height，归一化到0-1）。支持COCO、VOC等，通过Roboflow或LabelImg标注。

10. **预训练权重**：在COCO数据集上预训练，直接用于迁移学习（fine-tune），大大减少自家数据需求。

11. **数据增强（Augmentation）**：内置Mosaic、HSV颜色调整、翻转、缩放等，提升模型泛化能力，尤其适合蔬菜健康识别这种变异场景。

12. **评价指标**：mAP（mean Average Precision，主指标）、Precision、Recall、F1。Val模式会自动输出详细报告和混淆矩阵。

13. **训练命令示例**：`yolo detect train model=yolo11n.pt data=your_dataset.yaml epochs=100 imgsz=640`。支持resume断点续训。

14. **超参数（Hyperparameters）**：epochs（训练轮数）、batch（批大小）、imgsz（图像尺寸640/1280等）、lr（学习率）等，可通过yaml或命令行调整。

15. **YOLOv8 vs YOLO11**：YOLO11通常参数更少（约-22%）、精度更高（mAP提升）、小目标和效率更好，适合大多数新项目。

16. **部署友好**：支持导出到手机、边缘设备（Raspberry Pi、Jetson）、Web、云。TensorRT导出后速度极快。

17. **可视化结果**：Predict模式自动绘制边界框、标签、置信度；支持保存视频、生成热图等。

18. **社区与资源**：官方文档（docs.ultralytics.com）极佳，有大量guides和examples；Roboflow提供标注+数据集；Hugging Face也有集成。

19. **常见应用场景**：安防、自动驾驶、农业监测（植物疾病检测）、工业质检、医疗影像等。你的鱼菜共生蔬菜健康识别非常适合（检测叶片病斑/健康状态）。

20. **学习路径建议**：先跑官方预训练demo → 用Roboflow数据集训练一个自定义模型 → 调整超参观察效果 → 导出部署 → 进阶多任务/跟踪。遇到问题先查官方文档和GitHub Issues。

**快速上手建议**：安装后直接运行 `yolo detect predict model=yolo11n.pt source=0`（打开摄像头实时检测），感受实时性。结合你的蔬菜项目，从YOLO11n或s开始fine-tune即可。
